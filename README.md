# PindejosBowling
Bowling hub for the Pindejos Bowling League

## Local Development
First, you must `brew install node` so that you can run `node`/`npm` locally, and then you can `npm run dev` to start a local server, accessible at http://localhost:5173/PindejosBowling/

This way you can test/validate changes to the UI locally without having to push untested changes to `main` to trigger the deployment process
